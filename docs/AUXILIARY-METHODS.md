# Auxiliary Methods

These methods are available from the commandline.

__`create-random`__  
Creates a random alphanumeric string (lowercase) of configurable length.  
The default length is 15.

__`to-timestamp`__  
Returns the current datetime into ISO 8601 format.
Expects UTC.
